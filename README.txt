#------------------------------------------------------------------------------#
                         Movie Recommender System:
#------------------------------------------------------------------------------#
This is an hobby project, which is implemented in order to get an hands-on with
recommender system. MovieLens data set is used in this project.
#------------------------------------------------------------------------------#


#------------------------------------------------------------------------------#
Description Of Files And Directories:
#------------------------------------------------------------------------------#
readme.txt: 
  this file.

rm_pycache.sh: 
  simple script to delete .pyc files and __pycache__ directories

recommender.py: 
  python file which calls the main interface function of our machine learning
  system

package:
  directory which contains all the python source files which implements our
  movie recommender system 

movie_data_set:
  directory which contains MovieLens data set
#------------------------------------------------------------------------------#


#------------------------------------------------------------------------------#
Usage:
#------------------------------------------------------------------------------#
command line input:
  python recommender.py
#------------------------------------------------------------------------------#
